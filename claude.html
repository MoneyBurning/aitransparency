Frequently Asked
Why did I get blocked even though my session bar wasn't full?
Claude has two separate caps: a 5-hour rolling window and a weekly cap. Hitting the weekly cap locks you out even with session headroom remaining.
Can I see my exact usage on Claude?
Settings → Usage shows progress bars for both limits, but Anthropic never publishes exact token numbers — only relative multipliers (Pro = 1x, Max = 5x or 20x).
Is Claude's current 50% higher weekly limit permanent?
No. It's a promotion running through July 13, 2026. Expect limits to revert to standard baselines after that date.
Do Claude Code and claude.ai share the same usage pool?
Based on community reports, yes — Claude Code sessions draw from the same quota as claude.ai chat, which can exhaust limits faster for developers.
